DRAFT :: Jenkins的基本使用方法

2 May, 2017 · Read in about 1 min · (52 Words)
dev tool

安装

Jenkins的安装非常简便

在Ubuntu 16.04上安装：

wget -q -O - https://pkg.jenkins.io/debian-stable/jenkins.io.key | sudo apt-key add -
deb https://pkg.jenkins.io/debian-stable binary/
sudo apt-get update
sudo apt-get install jenkins

在CentOS或RHEL上安装：

sudo wget -O /etc/yum.repos.d/jenkins.repo https://pkg.jenkins.io/redhat-stable/jenkins.repo
sudo rpm --import https://pkg.jenkins.io/redhat-stable/jenkins.io.key
yum install jenkins

有一些插件可能还需要安装Docker才能使用，可以按照如下方式安装Docker：

  • Ubuntu
  • CentOS

初始化

在安装完成以后，需要进行初始化后才能使用。

初始化时，需要输入密码进行验证，获取密码方法（在安装Jenkins的主机上）：

cat /var/lib/jenkins/secrets/initialAdminPassword

之后，Jenkins会提示创建一个管理员用户才能进入。

然后Jenkins会提示进行安装一些插件，可以选择安装所有推荐插件也可以自定义选择进行安装即可。
